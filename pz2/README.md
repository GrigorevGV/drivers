# Практическое задание 2

## Задание
Создать простой модуль ядра Linux, который выводит сообщение при загрузке (инициализации) и выгрузке (удалении) модуля.

## Компиляция
```bash
make
```

## Загрузка модуля
```bash
sudo insmod simple.ko
```

## Проверка загрузки
```bash
lsmod | grep simple
dmesg | tail -5
```

## Выгрузка модуля
```bash
sudo rmmod simple
```

## Просмотр информации о модуле
```bash
modinfo simple.ko
```
