# Практическое задание 3

## Задание
Создать символьный драйвер (character device driver) и приложение для работы с ним. Драйвер регистрирует символьное устройство в системе, а приложение открывает это устройство через специальный файл.

## Компиляция драйвера
```bash
make
```

## Загрузка драйвера
```bash
sudo insmod driver.ko
```

## Проверка major номера
```bash
cat /proc/devices | grep foo
```

## Создание специального файла устройства
```bash
sudo mknod /dev/foo_device c <MAJOR> 0
sudo chmod 666 /dev/foo_device
```
Замените `<MAJOR>` на номер, полученный из предыдущей команды.

## Компиляция приложения
```bash
gcc -o main main.c
```

## Запуск приложения
```bash
sudo ./main
```

## Просмотр логов ядра
```bash
dmesg | tail -10
```

## Выгрузка драйвера
```bash
sudo rmmod driver
```
