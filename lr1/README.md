# Лабораторная работа 1

## Задание

1. Написать символьный драйвер с операциями `read`, `write`, `ioctl`
2. Написать тестовое приложение, читающее и пишущее `int`
3. В драйвере измерить время между чтением и записью
4. По `ioctl` в драйвере выдавать гистограмму при помощи `printk`

**Параметры гистограммы:**
- 20 бинов
- Размер бина: 50 микросекунд
- Нормализация не требуется

## Компиляция

```bash
make
```

## Загрузка драйвера

```bash
sudo insmod driver.ko
```

После загрузки автоматически создастся файл устройства `/dev/lr1_device`.

## Запуск тестового приложения

```bash
sudo ./app
```

## Просмотр гистограммы

```bash
sudo dmesg | tail -30
```

Или для фильтрации только гистограммы:
```bash
sudo dmesg | grep -A 25 "Histogram"
```

## Выгрузка драйвера

```bash
sudo rmmod driver
```
