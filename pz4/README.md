# Практическое задание 4

## Задание
Создать символьный драйвер, который:
- Автоматически создает специальный файл устройства
- Реализует чтение и запись в глобальный буфер
- Реализует ioctl для сброса содержимого буфера и получения информации, есть ли в нём данные

## Компиляция
```bash
make
```

## Загрузка драйвера
```bash
sudo insmod driver.ko
```

## Запуск приложения
```bash
sudo ./main
```

## Просмотр логов
```bash
sudo dmesg | tail -40
```

## Выгрузка драйвера
```bash
sudo rmmod driver
```
